- [Minimum Add to Make Parentheses Valid](https://leetcode.com/problems/minimum-add-to-make-parentheses-valid/)

```java
class Solution {
    public int minAddToMakeValid(String s) {
        int l = 0;
        int r = 0;
        int answer = 0;

        for (int i = 0; i < s.length(); i++) {
            if (s.charAt(i) == '(') {
                l++;
            } else {
                r++;
            }

            if (l < r) {
                answer++;
                l++;
            }
        }

        return answer + Math.abs(r - l);
    }
}
```
